# 👋 Xin chào, mình là Trương Hữu Nghĩa!

🎓 Mình là sinh viên yêu thích công nghệ và luôn tìm kiếm cơ hội để học hỏi những kiến thức mới về lập trình.

## 🚀 Giới thiệu

* 💻 Đang học và phát triển kỹ năng lập trình C, Git và GitHub.
* 📚 Quan tâm đến phát triển phần mềm, thuật toán và các dự án mã nguồn mở.
* 🎮 Đam mê các tựa game MMORPG, game khám phá và phiêu lưu, đặc biệt trên Roblox.
* 💪 Thường xuyên tập gym và quan tâm đến dinh dưỡng để duy trì sức khỏe.

## 🛠️ Kỹ năng

* **Ngôn ngữ:** C (đang học), HTML/CSS (cơ bản)
* **Công cụ:** Git, GitHub, Visual Studio Code
* **Đang học:** Cấu trúc dữ liệu, Thuật toán và phát triển phần mềm

## 🌱 Mục tiêu

* Nâng cao kỹ năng lập trình và tư duy giải quyết vấn đề.
* Xây dựng nhiều dự án thực tế để phát triển kinh nghiệm.
* Đóng góp cho các dự án mã nguồn mở trong tương lai.
* Không ngừng học hỏi các công nghệ mới.

## 📌 Sở thích

* 🎮 Chơi game khám phá, MMORPG và phiêu lưu.
* 🍜 Khám phá những món ăn ngon.
* 🏋️ Tập gym và rèn luyện sức khỏe.
* 🎧 Nghe nhạc khi học tập và làm việc.

## 📫 Kết nối

Nếu bạn muốn trao đổi về lập trình, công nghệ hoặc game, đừng ngần ngại kết nối với mình.
